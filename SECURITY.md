# Security Policy

## Supported Versions

- &#10003;  - Supported
- &#8764; - Running out of support
- &#10005; - Not supported

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | &#10003;           |

## Reporting a Vulnerability
You can report a vulnerability:
- Directly to a developer with in the Discord server
- By [creating an issue](https://github.com/WeatherStack/core/issues/new)
- By sending an email (which you can find via the
  [contact page](https://wtdawson.info/contact/v2/), this email inbox is checked regularly,
  but emails could be caught in spam filters, other options may be better).
